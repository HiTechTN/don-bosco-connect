✨ Fonctionnalités Principales

🤖

IA Adaptative

Apprentissage personnalisé avec Ollama. Le système ajuste le niveau automatiquement selon vos performances.

🌍

Multilingue

Disponible en Français, Anglais et Arabe (RTL supporté). Interface adaptée à chaque culture.

👥

Gestion des Rôles

Admin, Professeur, Élève, Parent - Chaque rôle a son espace dédié et ses fonctionnalités.

📱

Multi-Plateforme

Web (React), Mobile (React Native) - Une expérience unifiée partout.

📊

Suivi en Temps Réel

Tableaux de bord, statistiques, quiz interactifs et suivi de progression pour tous les utilisateurs.

🔧

Open Source

Code transparent, auto-hébergeable, Docker-ready. Contribuez sur GitHub !

⚙️ Installation Rapide

# Cloner et installer avec Docker (recommandé)

git clone https://github.com/HiTechTN/don-bosco-connect.git

cd don-bosco-connect

make install


# Ou avec gh CLI

gh repo clone HiTechTN/don-bosco-connect

cd don-bosco-connect && make install
