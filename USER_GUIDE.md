# Don Bosco Connect - Guide de l'Utilisateur

## 🎓 Bienvenue!

Don Bosco Connect est la plateforme éducative de l'école Don Bosco Tunis. Cette guide vous accompagne pour utiliser l'application selon votre rôle.

## 🚀 Accès à l'Application

### Web & Desktop
- **URL** : https://donbosco.tn (ou localhost:5173 si local)
- **Desktop** : Télécharger depuis [Releases](https://github.com/HiTechTN/don-bosco-connect/releases)

### Mobile
- **iOS/Android** : Scanner QR code sur https://donbosco.tn/mobile
- **APK** : Télécharger depuis Releases

---

## 👑 Tableau de Bord Admin

### Accueil
- Voir les statistiques globales (élèves, profs, classes)
- Alertes de décrochage en temps réel

### Gestion
1. **Élèves** : Ajouter/Modifier/Supprimer
2. **Professeurs** : Attribution matières et classes
3. **Classes** : Créer et configurer

### Analytiques
- Graphiques de réussite par classe
- Prédictions de risque (IA)

---

## 👨‍🏫 Espace Professeur

### Mes Cours
1. Créer un cours : `Cours → Nouveau Cours`
2. Ajouter leçons : Vidéos, PDF, Texte
3. Publier quand prêt

### Devoirs & Quiz
- **Créer un devoir** : `Devoirs → Nouveau`
- **Quiz IA** : `IA → Générer Quiz` (différencié selon niveau)
- **Correction** : Voir soumissions et noter

### Assistant IA
- `IA → Résumé de cours`
- `IA → Prévoir difficultés élèves`
- `IA → Quiz différencié`

---

## 🎓 Espace Élève

### Mon Apprentissage
- **Cours** : Suivre le fil conducteur
- **Progrès** : Voir XP, niveau, badges
- **Devoirs** : Soumettre avant échéance

### Mentor IA (24/7)
1. `IA → Poser une question`
2. L'IA répond basé sur vos cours
3. Formule : `Niveau = (Score × 0.6) + (Vitesse × 0.4)`

### Gamification
- 🏅 **XP** : Gagnez des points à chaque réussite
- 🎖 **Badges** : Débloquez des succès
- 📊 **Niveaux** : Progresser de 1 à 10

---

## 👪 Espace Parent

### Suivi de Mon Enfant
- **Notes** : Voir bulletins en temps réel
- **Présence** : Historique absences
- **Progrès** : XP, badges, niveau

### Communication
- **Messages** : Contacter enseignants
- **Insights IA** : Conseils personnalisés

---

## 🌐 Multi-Langues

Changer de langue :
- **Web** : Clic sur 🌐 en haut à droite
- **Mobile** : Paramètres → Langue

Langues supportées : 🇬🇧 English | 🇫🇷 Français | 🇹🇳 العربية

---

## 🔧 Dépannage

| Problème | Solution |
|----------|-----------|
| Je ne peux pas me connecter | Vérifier identifiants avec admin |
| Vidéo ne charge pas | Vérifier connexion, réessayer |
| IA ne répond pas | Vérifier que Ollama fonctionne (admin) |
| Application lente | Vider cache, redémarrer |
