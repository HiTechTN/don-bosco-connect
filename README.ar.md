## 🏗️ البنية

```
                     ┌──────────────────────┐
                     │  React 18 + Vite      │  ← ويب (SPA)
                     │  React Native (Expo)  │  ← جوال
                     └──────────┬───────────┘
                                │ HTTP / WebSocket
                     ┌──────────▼───────────┐
                     │  Nginx 1.25           │  :8080
                     │  تحديد المعدل · CSP   │
                     └──────────┬───────────┘
                                │
               ┌────────────────┼────────────────┐
               │                │                │
     ┌─────────▼──────┐  ┌─────▼──────┐  ┌──────▼──────┐
     │  FastAPI        │  │  Redis 7   │  │  Celery      │
     │  SQLAlchemy 2.0 │  │  تخزين     │  │  العمال      │
     │  JWT + MFA      │  │  طابور     │  └─────────────┘
     └─────────┬───────┘  └───────────┘
               │
     ┌─────────▼──────────────────────┐  ┌──────────────────────┐
     │  PostgreSQL 16 + pgvector       │  │  Ollama (خارجي)     │
     │  البيانات · تضمينات الذكاء      │  │  DeepSeek 14B       │
     └─────────────────────────────────┘  │  nomic-embed-text   │
               │                          └──────────────────────┘
     ┌─────────▼──────────┐
     │  MinIO (S3)         │
     │  الدروس · المستندات │
     └─────────────────────┘
```

### الخدمات

| الخدمة | الصورة | المنفذ | الدور |
|--------|--------|--------|-------|
| `db` | `pgvector/pgvector:pg16` | 5432 | قاعدة + متجهات |
| `redis` | `redis:7.2-alpine` | 6379 | تخزين مؤقت / طابور |
| `minio` | `minio/minio` | 9000-9001 | تخزين الملفات |
| `api` | *مخصص* | 8000 | الواجهة الخلفية FastAPI |
| `worker` | *مخصص* | — | مهام Celery |
| `frontend` | *مخصص* | 80 | بناء ثابت nginx |
| `nginx` | `nginx:1.25-alpine` | **8080** | وكيل عكسي |

> **Ollama** : خارجي (قابل للتكوين عبر `OLLAMA_BASE_URL`). النماذج الموصى بها: `deepseek-r1-tool-calling:14b`، `nomic-embed-text`.

## 📦 التثبيت

### تلقائي (مُوصى به)

```bash
curl -sSL https://raw.githubusercontent.com/HiTechTN/don-bosco-connect/main/scripts/install.sh | bash
```

### وضع GHCR (صور مُعدة مسبقاً)

```bash
USE_GHCR=1 GHCR_TOKEN=$(gh auth token) curl -sSL https://raw.githubusercontent.com/HiTechTN/don-bosco-connect/main/scripts/install.sh | bash
```

### يدوي

```bash
git clone https://github.com/HiTechTN/don-bosco-connect.git
cd don-bosco-connect
./scripts/setup.sh    # تكوين .env، SSL، المفاتيح
./scripts/start.sh    # تشغيل Docker
```

## 💻 التطوير

```bash
# الواجهة الخلفية FastAPI
cd backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
pytest -v

# الواجهة الأمامية (React + Vite)
cd frontend
npm install && npm run dev    # → http://localhost:5173

# الجوال (React Native / Expo)
cd mobile
npm install && npx expo start

# إنشاء فيديو تجريبي
pip install moviepy gtts pillow numpy
python demo/generate_demo_video.py  # → demo/demo_don_bosco.mp4
```

**توثيق API** → [`http://localhost:8000/docs`](http://localhost:8000/docs)
