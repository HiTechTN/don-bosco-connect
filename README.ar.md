## 🚀 نظرة عامة

**دون بوسكو كونكت** هي منصة تعليمية شاملة تحول تجربة التعلم في مدرسة دون بوسكو تونس. تجمع بين الإدارة الأكاديمية الصارمة والتعاون البيداغوجي والتعلم المخصص بالذكاء الاصطناعي.

### ✨ المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 🤖 **التعلم التكيفي** | محتوى بالذكاء الاصطناعي يتكيف مع مستوى كل تلميذ |
| 👨‍🏫 **التعاون** | ربط التلاميذ والمعلمين وأولياء الأمور |
| 🎮 **العاب التعليمية** | نقاط الخبرة والشارات والمستويات |
| 📊 **التحليلات المتقدمة** | تتبع مفصل وتنبيهات استباقية |
| 📱 **منصات متعددة** | الويب والكمبيوتر والموبايل |

## 🔧 التقنيات

| المكون | التقنية |
|--------|---------|
| **الواجهة الخلفية** | جانغو + DRF |
| **قاعدة البيانات** | PostgreSQL + pgvector |
| **ذكاء اصطناعي** | Ollama (Qwen 2.5) |
| **ويب** | React + Electron |
| **موبايل** | React Native (Expo) |

---

## 🚦 التشغيل

```bash
# الواجهة الخلفية
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

# الويب
cd web
npm install
npm run dev

# الموبايل
cd mobile
npx expo start
```
